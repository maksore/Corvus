## ✨ What is Corvus?

**Corvus** is more than just a Discord bot—it's a sleek, intelligent companion designed to elevate your server experience. With a dark, modern aesthetic and custom emojis, Corvus seamlessly blends into any community while delivering powerful features.

Whether you're looking for fun games, a smart AI chat partner, high‑quality music playback, or robust moderation tools, Corvus has you covered. All commands work with both slash commands (`/`) and a customizable prefix (default `$`).

---

## 🎮 Features at a Glance

### 🤖 AI Chat
Have natural conversations with Corvus using the `/ask` command. The bot uses a responsive chatbot API to provide clever, context‑aware replies. *(Upgradable to OpenAI for even smarter interactions.)*

### 🎲 Games
Challenge your friends or pass the time solo with a variety of engaging games:
- **Dice Roll** – Roll a custom‑sided die.
- **Rock Paper Scissors** – Play against the bot with custom emojis.
- **Trivia** – Test your knowledge with timed questions from various categories.
- **Tic‑Tac‑Toe** – Interactive button‑based matches against other members.

### 🎵 Music
Corvus delivers crisp, high‑quality audio from YouTube directly to your voice channel.
- Play songs, playlists, or search by keywords.
- Queue system with skip, stop, and now‑playing displays.
- Clean embeds show track info and progress.

### 🛡️ Moderation
Keep your server safe and tidy with essential moderation tools:
- **Purge** – Delete messages in bulk.
- **Kick / Ban / Mute** – Manage troublesome members with proper permission checks.

### ⚙️ Customization
- **Per‑server prefix** – Set a custom prefix for your community (e.g., `?`, `!`, `c!`).
- **Sleek embeds** – Every response is wrapped in a dark‑themed embed with custom emojis for a polished look.

## 🚀 Getting Started

1. **Invite Corvus** – Click the **Add Corvus** button above and select your server.
2. **Set a prefix (optional)** – Use `/setprefix <new_prefix>` to customize the command prefix.
3. **Explore commands** – Type `/help` to see everything Corvus can do!

---

## 📋 Command Highlights

| Command | What it does |
|---------|--------------|
| `/help` | Shows all available commands with categories |
| `/ask <question>` | Chat with Corvus AI |
| `/play <song>` | Play a song in your voice channel |
| `/roll [sides]` | Roll a dice (default d6) |
| `/rps <choice>` | Rock, Paper, Scissors against the bot |
| `/trivia` | Answer a random trivia question |
| `/tictactoe @user` | Challenge a friend to Tic‑Tac‑Toe |
| `/purge <amount>` | Delete a number of messages |
| `/kick @user` | Kick a member from the server |
| `/ban @user` | Ban a member |
| `/mute @user <minutes>` | Timeout a member |
| `/setprefix <new>` | Change the bot's prefix for this server |

*All commands also work with the server's custom prefix.*
